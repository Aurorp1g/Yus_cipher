项目概述

YuS Cipher 是一个基于论文 "YuS: A FHE-friendly Stream Cipher Based on New Quadratic Permutations" 实现的高级流密码系统。该密码专为同态加密（FHE）环境设计，具有低乘法深度和高效的性能特点。

同态加密友好

专为BGV/BFV同态加密方案优化，支持128位安全级别，8192多项式度，适合在加密数据上直接计算。

高效FHE性能

在实际测试中实现3.13 KiB/s的同态加密吞吐量，评估时间仅需15毫秒，内存使用峰值34MB。

高安全性

基于新型二次排列，提供128位安全级别，通过16项单元测试验证，确保密码学安全性。

核心特性

密码学特性

  • 基于F_p域新型二次排列（p ≡ 2 mod 3）
  • 差分均匀性为 p^(n-1)，抵抗差分攻击
  • Walsh谱有界，抵抗线性攻击
  • 抵抗Gröbner基和XL代数攻击
  • SPN结构，12个并行S-box
  • 固定线性映射，避免随机矩阵开销

实现特性

  • C++17标准实现，跨平台兼容
  • 模块化架构，易于扩展和维护
  • 支持HElib(BGV)和SEAL(BFV)双方案
  • 全面的单元测试和集成测试
  • 详细的性能基准测试
  • 命令行工具和编程接口

系统架构

核心组件

YuSCipher

主密码类，提供加密/解密接口，支持36字节密钥和可配置安全级别。

QuadraticPermutation

二次排列实现，提供S-box功能和密码学性质验证。

SPNStructure

替代-置换网络结构，包含12个并行S-box和固定线性映射。

FHEWrapper

同态加密包装器，支持BGV和BFV双方案，提供统一的FHE接口。

代码示例

// 基本使用示例 #include "yus/yus_core.h" #include "yus/fhe_wrapper.h" int main() { // 初始化YuS Cipher YuSCipher cipher; std::vector<uint8_t> key(36, 0xAB); // 36字节密钥 cipher.initialize(key); // 初始化FHE包装器 FHEWrapper fhe_wrapper; FHEParams params = {128, 8192, 300, FHE_SCHEME::BFV}; fhe_wrapper.initialize(params); // 加密数据 std::vector<uint8_t> plaintext = {0x01, 0x02, 0x03}; std::vector<uint8_t> ciphertext = cipher.encrypt(plaintext); return 0; }

使用方法

命令行工具

# 构建项目 (Windows) mkdir build && cd build cmake .. -DCMAKE_BUILD_TYPE=Release cmake --build . --config Release # 运行演示程序 ./yus_example.exe # 运行测试套件 ./yus_test.exe # 查看测试结果 # yus_example.exe 输出详细的FHE性能指标 # yus_test.exe 运行16个测试验证功能正确性

CMake配置

# CMakeLists.txt 关键配置 cmake_minimum_required(VERSION 3.15) project(yus_cipher VERSION 1.1.0) # 设置C++标准 set(CMAKE_CXX_STANDARD 17) set(CMAKE_CXX_STANDARD_REQUIRED ON) # 添加依赖库 find_package(HElib REQUIRED) find_package(SEAL REQUIRED) find_package(GTest REQUIRED) # 构建主库 add_library(yus STATIC src/yus_core.cpp src/fhe_wrapper.cpp) target_link_libraries(yus PRIVATE HElib::helib SEAL::seal)

FHE参数配置

// 当前最优参数配置 FHEParams fhe_params; fhe_params.security_level = 128; // 128位安全级别 fhe_params.poly_modulus_degree = 8192; // 8192多项式度 fhe_params.cipher_modulus_bits = 300; // 300位密文模数 fhe_params.scheme = FHE_SCHEME::BFV; // 使用BFV方案 // 初始化FHE包装器 FHEWrapper fhe_wrapper; fhe_wrapper.initialize(fhe_params);
